# Statistical Data Modeling and Probability Analysis

Bu çalışma, gerçek dünya veri setleri üzerinde istatistiksel yöntemler kullanarak verinin karakteristiğini belirleme ve olasılık dağılımlarını analiz etme üzerine kuruludur.

## Proje İçeriği
* **Olasılık Analizi:** Veri setinin Olasılık Yoğunluk Fonksiyonu (PDF) ve Kümülatif Dağılım Fonksiyonu (CDF) grafiklerinin oluşturulması.
* **Aykırı Değer Tespiti:** Z-score ve IQR yöntemleri kullanılarak veri setindeki aykırı değerlerin (outliers) belirlenmesi ve temizlenmesi.
* **İstatistiksel Modelleme:** Farklı veri dağılımlarının matematiksel modellerle karşılaştırılması.

## Kullanılan Teknolojiler
* Python
* NumPy & SciPy
* Matplotlib

## Nasıl Çalıştırılır?
`Statistical_Inference_and_Probability_Modeling.ipynb` dosyasını Jupyter Notebook üzerinde çalıştırarak analiz adımlarını takip edebilirsiniz.
